ГорбенкоТех - Москва. Вид с небоскрёба.

May 15, 2014

2014   ·   ГорбенкоТех  

Немного моих фотографий с 58 этажа небоскрёба Imperia Tower в Москва-Сити. На фотографиях - Останкинская башня, стадион Лужники, Ценр Международной Торговли (Хаммеровский центр), Кутузовский проспект и жилой дом Андропова-Брежнева на нем, вид на Меркурий-тауэр. Хороший был день!

Блог из архива. Этот хороший день был в далёком 2014 году.
